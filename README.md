# EU Tech Sovereignty — Interactive Explorer

An interactive, single-file static website explaining the European Union's tech
sovereignty initiatives: how they build on each other, their dependencies, what
companies must do, what they mean for end users, and a timeline.

## Contents
- `index.html` — the interactive site (also `eu-tech-sovereignty.html`, identical)
- `Management Summary.md` — overview of the whole agenda
- `Initiatives/` — one summary per initiative (19 in total)
- `Source Documents Index.md` — direct links to the official EU papers/laws
- `.nojekyll` — tells GitHub Pages to serve files as-is (no Jekyll processing)

## Deploy on GitHub Pages
1. Push this folder to a GitHub repository.
2. Repo **Settings → Pages → Build and deployment → Source: Deploy from a branch**.
3. Select `main` branch, `/ (root)`, save.
4. The site goes live at `https://<username>.github.io/<repo>/`.

The page is fully self-contained (inline CSS/JS, no external libraries or build
step), so it runs anywhere static files are served.

## Note
Independent explanatory tool, not an official EU publication. Dates for proposed
measures may change; consult the linked official documents for binding text.
